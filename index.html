About Me

Hi! I'm Stewart Milton, a passionate web developer with over 1 year of experience in creating dynamic and responsive websites. I specialize in front-end development but have a solid understanding of back-end technologies as well. My journey in web development started when I was in college, where I discovered my love for coding and design.

Throughout my career, I've had the opportunity to work on various projects, ranging from simple landing pages to complex web applications. My expertise lies in:

  • HTML & CSS: Crafting semantic, clean, and responsive code.
  • JavaScript & jQuery: Bringing interactivity and dynamic features to websites.
  • Bootstrap: Utilizing the power of Bootstrap to build mobile-first, responsive designs.
  • Version Control: Proficient in using Git and GitHub for collaboration and version management.

I'm always eager to learn new technologies and improve my skills. Recently, I've been exploring frameworks like React and Vue.js to further enhance my front-end capabilities.

In my free time, I enjoy contributing to open-source projects, writing technical blogs, and staying up-to-date with the latest industry trends. When I'm not coding, you can find me exploring new hiking trails or experimenting with photography.
